📋 ÉTAPE UNIQUE - Ajouter 1 ligne dans index.html
───────────────────────────────────────────────────────────────────────────

1. Ouvrez le fichier: index.html

2. Cherchez la ligne (environ ligne 4569):
   <script src="https://cdn.jsdelivr.net/npm/chart.js"></script>

3. Ajoutez JUSTE AVANT cette ligne:
   <script src="dropdown.js"></script>

4. Le résultat final devrait être:
   
   <script src="dropdown.js"></script>
   <script src="https://cdn.jsdelivr.net/npm/chart.js"></script>

5. Sauvegardez le fichier

   EN CAS DE PROBLÈME
═══════════════════════════════════════════════════════════════════════════

Si les dropdowns ne s'affichent pas:

1. Vérifiez que dropdown.js est dans le même dossier que index.html
2. Ouvrez la console (F12) et vérifiez qu'il n'y a pas d'erreur
3. Vérifiez que la ligne <script src="dropdown.js"></script> est bien ajoutée
4. Rechargez la page (Ctrl+F5 pour vider le cache)
